QChip Vue 组件基本上是一个简单的 UI 实体块，以紧凑的方式展示更复杂的数据，例如用来展示联系人。

QChip 可以包含多种内容：头像、文案、图标。还能被设置成可关闭或可删除的。


::: tip
也看看 [QBadge](/vue-components/badge).
:::
